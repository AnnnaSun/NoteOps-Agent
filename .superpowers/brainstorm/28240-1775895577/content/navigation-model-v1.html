Global Navigation Rule
Home Notes Ideas Workspace Trend Inbox
所有页面共享一层极简顶栏导航，用户可随时切换内容域。
但每个页面主体只展示单一内容，不再混合多个大区块。

本段设计结论

结构

  • `Home` 是入口页，不再承担重型工作台职责。
  • `Notes` / `Ideas` 属于 library pages。
  • `Workspace` / `Trend Inbox` 属于 action pages。
  • 全局只有一层主导航，不引入复杂层级和 breadcrumbs。

边界

  • 首页不出现具体内容预览。
  • `Trend Inbox` 不承载 Note / Idea 详情内容。
  • 这一步不做多层页面体系。
